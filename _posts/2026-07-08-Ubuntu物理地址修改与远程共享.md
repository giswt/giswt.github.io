<font size=3 color=Red>操作系统：Ubuntu 22.04.5 LTS</font>    

### 简单描述
希望 Ubuntu能够修改物理网卡的MAC地址，并能够通过Windows远程桌面访问
### MAC地址修改

使用以下任一命令查看当前的网卡及MAC地址  
```
ifconfig  #单独使用

ip address # 单独使用
```  
然后 使用命令将要修改的网卡 禁用  修改地址 启用  
```  
sudo ip link set dev wlp3s0 down  
sudo ip link set dev wlp3s0 address xx:xx:xx:xx:xx:xx
sudo ip link set dev wlp3s0 up

```
